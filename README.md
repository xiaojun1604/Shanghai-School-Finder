# 上海闵行选校购房助手 (Minhang School & House Finder)

主要用于解决家有学童（转学）在上海闵行区的选校与购房规划问题。特别针对**集体户口**、**500万预算**、**二梯队目标**的家庭。

## 核心功能
1. **名校筛选**：基于500万总价预算，筛选闵行区优质公办二梯队学校。
2. **政策导向**：重点解读“集体户”转学优先级及“人户一致”的重要性。
3. **购房建议**：针对每个学校提供对口板块的典型房源及估价。

## 闵行二梯队学校推荐
- **闵行实验小学（春城校区）**：二梯队翘楚，对口初中极强。
- **莘城学校**：九年一贯制，省心之选。
- **航华一小**：紧邻紫藤社区，转学便利。
- **平阳小学**：对口实验西校，古美核心。
- **田园外小**：上升势头强劲，性价比高。

## 使用说明
1. 访问 `index.html` 即可查看。
2. 应用为静态网页，无需后台。

## 注意事项
> [!IMPORTANT]
> 政策每年更新，购房前请务必咨询当地教育局，并让中介核实“五年一户”学位锁定情况。
